BookSave

BookSave is a web app that help users find inspiration for reading books. A place where you can search and save different books in a "To-read" list. After you have finished reading a book, it can be saved in a so called "Finished" list. You can grade the book based on how much you liked it.

This project was made with React.js. BookSave uses Google Books API.

MusicForLater

A service used to make life easier for music lovers. Through this service, users can see all channels from Sveriges Radio, the songs that are being played and then find these songs on Spotify. The API is primarily created for web applications. For backend, the project used Python with Flask and Flask_Session. The frontend technologies were HTML, CSS and Javascript (jQuery).

Foodnest

First ever school project. A food-related web application with the aim of letting users inspire each other by sharing recipes. A place for those who find it fun to create and cook their own as well as others' meals in their everyday life.

We used HTML, CSS, JS, Python (Bottle) and MySQL for this project.
